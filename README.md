# analise_exploratoria_DataScience_Alura
Análise exploratória de dados de filmes com pandas, seaborn e matplotlib (MovieLens + TMDB)

# Análise de Dados de Filmes

Projeto do curso de Introdução a Data Science da Alura.

## Sobre

Análise exploratória usando dados do MovieLens (notas de filmes) e do TMDB (orçamento, receita, idioma).

## Ferramentas
- Python
- pandas
- seaborn
- matplotlib

## O que foi feito
- Média, mediana e desvio padrão das notas
- Gráficos de distribuição (histograma e boxplot)
- Comparação de notas entre filmes
- Análise de idioma dos filmes

## Arquivo
- `notas.ipynb` — notebook com a análise
